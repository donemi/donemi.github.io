# [《图书馆的折角》](https://donemi.github.io/)

市图书馆哲学区第三排，那本《存在与虚无》的第127页，有个微小折角。

周晓发现的。那时她刚离婚，像被掏空的贝壳，来图书馆只为听翻书声——证明世界还在有序运转。

折角很克制，只把右上角向内折了毫米。她抚平它，指尖传来纸张的记忆：凉，滑，带点抗拒。下一页，铅笔写的日期：“2009.3.14”。字迹清瘦，像雪地上的鸟爪印。

她忽然好奇：十五年前的这个午后，是谁在这里读萨特？是男是女？为什么折页？又为什么没回来？

从此她每周三下午都来。总先抚平折角，仿佛那是约定。书架成了她的告解室，透过书与书的缝隙，能看见人生百态：备考的学生、消磨时光的老人、躲雨的情侣。但折角主人再没出现。

直到梅雨季，空气能拧出水。她又在哲学区，听见脚步声停在身后。

“抱歉，”男声温和，“您挡着《悲剧的诞生》了。”

他抽书时，相邻的《存在与虚无》倒下。两人同时伸手，手指在书脊上轻触。他先缩回手：“对不起。”

“没关系。”她注意到他无名指根有圈白痕——刚摘婚戒的痕迹。

后来总遇见。他叫沈河，在大学教美学。也离婚，也来图书馆寻找秩序。他们开始坐同一张长桌，各读各的书，像两座相邻但不相连的岛屿。

某天，沈河借了那本《存在与虚无》。还回来时，周晓发现第127页夹着银杏书签。

“我见过那个折角。”沈河说，像承认罪行，“很多年前折的。那时觉得这页写得真好，想下周来抄。结果再没找到时间。”

“现在呢？”

“现在觉得，”他翻开书，“萨特说得对，人确实是自由的。只是我们常忘记如何使用自由。”

秋天，他们一起抚平了那个折角。纸张发出叹息般的轻响。

转折在初雪那天。沈河没来。下周、下下周也没来。她查借阅记录：他借的书超期了。打他留的电话，空号。

周晓第一次感到恐慌。不是失去谁的恐慌，而是故事突然中断的恐慌。她开始在他常坐的位置等他，像等一艘忘记归期的船。

元旦前一天，闭馆音乐响起时，管理员递来一封信。

“沈先生留给您的。他特意说，如果一个月后他没来取回，就交给常坐在哲学区的女士。”

牛皮纸信封，里面只有一张便签：

“周晓，原谅我不告而别。前妻确诊癌症，我回去照顾她。这不是牺牲，是清债。你曾问我为什么总读悲剧——现在我知道了，悲剧不是痛苦，是选择。而我选择回去。

《存在与虚无》第127页，萨特说：‘人是自己行动的总和。’我的行动里，包括认识你。

如果可能，请继续抚平那些折角。在某个被遗忘的书页里，也许藏着另一个人等待被理解的十五年。”

信纸右下角，有个新的折角。她抚平它，动作很轻，像触碰正在愈合的伤口。

窗外飘雪了。周晓把信夹回第127页。合上书时她想：有些折角不需要抚平，它们是书签，标记着我们曾被怎样地阅读过。

她没再等沈河。但每周三仍来，坐在老位置。有时帮管理员整理书架，把被折角的书页一一抚平。这个动作渐渐有了新含义：不是抹去痕迹，是承认那些痕迹曾存在过。

春天，她在《悲剧的诞生》里发现压干的樱花。秋天，《查拉图斯特拉如是说》中飘出银杏书签。她收集这些时间的信物，像收集潮水退去后留在沙滩上的贝壳。

又一个雨季，她在《纯粹理性批判》里发现铅笔字：“今天决定活下去。2009.3.14。”

是同一天。同一个人。

周晓终于哭了。为所有在图书馆寻求答案的灵魂，为所有折了角却没能回来的人生，为所有在书页间传递的、静默的陪伴。

闭馆时，她把一张新便签夹进《存在与虚无》：

“沈河，我明白了。折角不是中断，是呼吸。我们都在各自的页码里活着，被某个人阅读，或等待被阅读。这就够了。”

她抚平书页，最后一个离开。脚步声在长廊回荡，像心跳，像翻书声，像所有未完成的故事正在被续写。
